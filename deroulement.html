Le déroulé de l'atelier

Quatre étapes pour transformer une feuille blanche en affiche de festival — puis en musique.

Étape 1

Devine le style !

L'animateur montre une affiche exemple. À l'aide de la banque de mots ci-dessous, l'équipe doit deviner de quel style graphique il s'agit.

📖 Banque de mots — tous les styles possibles
Étape 2

Écris ta consigne, avant de générer

Avant de lancer la génération de l'affiche A3, l'équipe doit être précise : le style, le contenu, et la date du festival (attention aux jours fériés !).

💡 Vérifie que la date n'est pas un jour férié — sauf si c'est fait exprès pour l'histoire du festival !

⚠️ Plus la consigne est précise (style + contenu + date + lieu + ambiance), plus l'affiche générée ressemblera à ce que l'équipe imagine. Vous avez droit à 3 générations maximum.

Remplis les champs ci-dessus pour voir ta consigne se construire ici.
Étape 3

Choisis ton affiche parmi 3 générations

L'animateur génère 3 versions de l'affiche à partir de la consigne. Dépose chaque image dans une case ci-dessous : elles s'affichent côte à côte pour comparer.

🖨️ Attention, comme à l'imprimerie : une fois qu'une image est cliquée pour être validée, c'est définitif. Impossible de revenir en arrière, alors regardez bien toutes les propositions avant de choisir !
Étape 4

Crée ta musique

Dernière étape : composer une musique qui correspond à l'ambiance de l'affiche validée. Le style graphique doit se retrouver dans le style musical !
